# 🌙 Eid Digital Card Studio (استوديو بطاقات عيد الأضحى)

A premium, interactive web application that allows users to easily create and personalize beautiful Eid Al-Adha greeting cards.

![Eid Digital Card Studio](frontend/public/assets/templates//logo.png)

## 🌟 Concept

The Eid Digital Card Studio provides a sleek, modern, and user-friendly interface to generate high-quality Eid greeting cards. Users can select from various professionally designed templates, customize the sender and recipient names, and add a personalized message or Dua. The application features a real-time live preview with elegant Arabic typography, and allows users to instantly download the final card as a high-resolution PNG ready for sharing on social media or WhatsApp.

## ✨ Features

- **Live Preview:** See changes in real-time as you type names or messages.
- **Multiple Premium Templates:** Choose from different aesthetic designs (modern dark, warm, classic).
- **Customization:** Personalize "To", "From", and the main greeting message.
- **Arabic Typography:** Carefully selected Arabic fonts for an authentic and elegant look.
- **Instant Download:** Export the customized card as a high-quality PNG image using HTML5 Canvas.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## 🛠️ Tech Stack

- **Frontend Framework:** React (with Vite)
- **Styling:** Tailwind CSS (for modern utility-first styling and glassmorphism effects)
- **Icons:** Lucide React
- **Image Generation:** Native HTML5 Canvas API

## 🚀 How to Run Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/eid-greeting-card.git
   cd eid-greeting-card/frontend
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173` to view the app.

## 📦 Build for Production

To create a production-ready build:

```bash
npm run build
```

The compiled files will be located in the `dist` folder.
